Come funziona Guscio

Guscio è un sistema semplice e robusto pensato per fornire istruzioni rapide, sicure e sempre disponibili tramite TAG NFC, QR code o link diretto.

1. Attivazione

Avvicina il telefono al TAG NFC oppure scansiona il QR. Il sistema apre automaticamente la pagina dedicata alle istruzioni.

2. Contenuti disponibili

• Istruzioni operative
• Procedure di sicurezza
• Video dimostrativi
• Documenti tecnici
• Collegamenti rapidi

3. Funzionamento offline

Se la rete è debole, Guscio mostra comunque le istruzioni essenziali grazie alla struttura HTML leggera e ottimizzata.

4. Aggiornamenti

Ogni modifica nel repository viene pubblicata automaticamente su GitHub Pages. Nessuna manutenzione richiesta sul TAG.
